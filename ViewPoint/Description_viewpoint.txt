Описание метрик Viewpoint
Метрика             Описание                                                              Тип поля
SESSION_ID          ID сессии                                                             NUMBER
USERNAME            Имя пользователя,запустившего запрос                                  CHARACTER
ΔCPU                Использованное процессорное время за последнюю минуту                 NUMBER
ΔI/O                Число логических операций ввода\вывода за последнюю минуту            NUMBER
IMPACT_CPU          Чем больше отличается от Delta CPU, тем более "перекошенным" 
                    является запрос, нагрузка по AMP-ам распределена неравномерно         NUMBER
SPOOL               Используемое дисковое пространство для выполнения запроса             NUMBER
DURATION            Как долго запрос был запущен                                          NUMBER
START               Когда запрос был запущен                                              NUMBER
BLOCKED TIME        Как долго запрос был заблокирован                                     NUMBER
IN STATE            Как долго запрос был в текущем состоянии                              NUMBER
WORKLOAD            WORKLOAD, в котором работает запрос, связано с разделением ресурсов
                    сервера                                                               CHARACTER
REQ CPU             Количество процессорного времени потребляемого за все 
                    время выполнения запроса                                              NUMBER
PJI                 PJI - Product Join Index отношение CPU к I/O, этот показатель 
                    становится высоким когда потребляется много процессоров и мало
                    дисков, например во время "продукт джойнов" или при работе UDF.       NUMBER
STATE               Текущее состояние запроса                                             CHARACTER
REQ I/O             Количество логических операций ввода / вывода за все
                    время выполнения запроса                                              NUMBER
I/O SKEW            Перекос по I/O показывает насколько равномерно по
                    ампам используются диски. Считается как 1-(среднее/максимальное).     NUMBER
ACCOUNT             Аккаунт, под которым был запущен запрос                               CHARACTER
PARTITION           Раздел, в котором выполняется запрос.Бывают partition :
                    SQL, CONSOLE, MONITOR. Все запросы работают в SQL, PMon 
                    и частично viewpoint в MONITOR.                                       CHARACTER
